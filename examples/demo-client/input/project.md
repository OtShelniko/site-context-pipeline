fictional company "Example Co" sells local and long-haul delivery services
on a fictional `example.com` domain.

These notes are reproduced verbatim in the agent context pack so a human
reviewer (or an LLM) can read them next to the inventory. Keep this file
short and factual. The pipeline does not parse it in 0.1.

- Tone: neutral, practical, no marketing superlatives.
